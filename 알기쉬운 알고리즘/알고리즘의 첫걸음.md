### 1.1 최대 숫자 찾기

#### 순차탐색

: 하나씩 숫자를 비교하면서 가장 큰 숫자를 기억하가며 숫자를 찾는 방법.

```python
def max(number_list):
    M = number_list[0]
    for number in number_list:
        if M < number:
            M = number
    return number        
```

***#나를 위한 공부로 해당 글을 올리는 것이기 때문에 몰랐던 내용, 부족한 부분을 위주로 자세히 기록할 예정.***
